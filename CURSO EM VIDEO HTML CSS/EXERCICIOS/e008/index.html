Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a TAG B (Não semântica - Sem sigificado)

Nesta frase temos um termo em destaque usando a TAG STRONG (Semântica- Com significado)

Itálico /Ênfase

Nesta frase temos uma termo em itálico usando a TAG I (Não semântico)

Nesta frase temos uma termo em ênfase usando a TAG EM (Semântico)

Texto marcado

Podemos também criar um texto marcado usando a tag MARK.

E nesse outro paragráfo temos outro texto marcado no final

Texto grande e pequeno.

Estamos criando um texto grande e um texto pequeno nesse paragráfo.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag U que é não semântica.

Texto sobrescrito

Para insersir coisas do tipo x2 + y5 = z9

Texto subscrito

Para inserir coisas do tipo H2SO4
